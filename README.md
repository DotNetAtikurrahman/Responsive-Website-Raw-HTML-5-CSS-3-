## 📖 Chapter Breakdown

### 🟠 HTML5 Chapters (1–8)

---

#### Chapter 1 — Getting to Know HTML
> `chapter1.html` · `chapter1.css`

The foundation. Covers the anatomy of an HTML document: character encoding, favicon linking, DOCTYPE declaration, and basic structural tags.

**Concepts:** Document structure · `<head>` & `<body>` · Meta tags · Favicon · Basic HTML elements

---

#### Chapter 2 — HTML Elements
> `chapter2.html` · `chapter2.css`

A comprehensive tour of HTML's built-in elements with live examples for every major content type.

**Concepts:** Headings (`h1`–`h6`) · Paragraphs · Hyperlinks · Images · Image maps · Responsive images · Ordered & unordered lists · Description lists · Tables · Line breaks · `<progress>` bar · Computer code tags · HTML entities

---

#### Chapter 3 — HTML Styles & Formatting
> `chapter3.html` · `chapter3.css`

Introduces how styling is applied and explores formatting elements that add semantic meaning to text.

**Concepts:** `<b>`, `<i>`, `<strong>`, `<em>`, `<mark>`, `<del>`, `<ins>`, `<sub>`, `<sup>` · Inline styling

---

#### Chapter 4 — Semantic HTML
> `chapter4.html` · `chapter4.css`

Demonstrates the importance of meaningful markup using HTML5's semantic elements and interactive disclosure widgets.

**Concepts:** `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>` · `<details>` & `<summary>` · `<figure>` & `<figcaption>`

---

#### Chapter 5 — HTML Web Forms
> `chapter5.html` · `chapter5.css`

Builds a fully structured contact form with a sidebar layout, covering every major form input type and attribute.

**Concepts:** `<form>`, `<input>`, `<textarea>`, `<select>`, `<button>` · Input types (text, email, password, date, radio, checkbox, file, range, color) · Form validation attributes · `<fieldset>` & `<legend>` · Side navigation layout

---

#### Chapter 6 — HTML Multimedia
> `chapter6.html` · `chapter6.css`

Embeds real audio and video files directly in the browser using native HTML5 media elements.

**Concepts:** `<audio>` (autoplay, controls, loop) · `<video>` (controls, poster, sources) · Media file formats

---

#### Chapter 7 — HTML Graphics
> `chapter7.html` · `chapter7.css`

An in-depth exploration of browser-native vector and raster graphics without any external libraries.

**SVG shapes:** Rectangle · Circle · Ellipse · Line · Polyline · Polygon · Path · Gradients (linear & radial)

**Canvas:** `<canvas>` element with JavaScript-based 2D drawing context

---

#### Chapter 8 — HTML Advanced
> `chapter8.html` · `chapter8.css`

Clarifies the fundamental difference between block-level and inline elements and how they affect document flow.

**Concepts:** Block elements (`<div>`, `<p>`, `<h1>`–`<h6>`) · Inline elements (`<span>`, `<a>`, `<img>`) · Default rendering behavior · The `<div>` container pattern

---

### 🔵 CSS3 Chapters (11–22)

---

#### Chapter 11 — Getting to Know CSS
> `chapter11.html` · `chapter11.css`

Introduces the three methods of applying CSS and explains the shorthand syntax and unit system.

**Concepts:** Inline CSS · Internal `<style>` · External stylesheet · CSS shorthand properties · Units: `px`, `%`, `em`, `rem`, `vh`, `vw`

---

#### Chapter 12 — CSS Selectors
> `chapter12.html` · `chapter12.css`

A thorough demonstration of every CSS selector category with interactive live examples.

**Concepts:**
- **Basic selectors:** Element, class, ID, attribute, universal (`*`)
- **Pseudo-classes:** `:hover`, `:focus`, `:active`, `:visited`, `:checked`, `:disabled`, `:required`
- **Structural selectors:** `:nth-child`, `:first-child`, `:last-child`, `:nth-of-type`
- **Pseudo-elements:** `::before`, `::after`, `::first-line`, `::first-letter`, `::selection`
- **Combinators:** Descendant, child (`>`), adjacent sibling (`+`), general sibling (`~`)

---

#### Chapter 13 — CSS Cascading
> `chapter13.html` · `chapter13.css`

Demonstrates how CSS resolves conflicts between competing rules through the cascade algorithm.

**Concepts:**
- Rule 1: Last declaration wins
- Rule 2: Higher specificity wins — inline > ID > class > element
- Rule 3: Inline styles override internal/external stylesheets
- Rule 4: `!important` overrides everything

---

#### Chapter 14 — CSS Box Model
> `chapter14.html` · `chapter14.css`

Provides a hands-on visualization of how every HTML element occupies space in the document.

**Concepts:** `content-box` vs `border-box` · Margin & margin collapsing · Padding · Outline · Border styles (solid, dashed, dotted, double, groove, ridge) · `border-radius` · `border-image`

---

#### Chapter 15 — CSS Color
> `chapter15.html` · `chapter15.css`

Explores the full spectrum of CSS color specification formats.

**Concepts:** Named colors · HEX (`#rrggbb` / `#rgb`) · RGB & RGBA · HSL & HSLA · `currentColor` · `transparent`

---

#### Chapter 16 — CSS Text
> `chapter16.html` · `chapter16.css`

A deep dive into typography and all the ways CSS controls how text is rendered, spaced, and positioned.

**Concepts:** `font-family`, `font-size`, `font-weight`, `font-style` · `text-decoration` (style, color, thickness) · `text-transform` · `white-space` · `text-overflow` (ellipsis) · `word-wrap` & `word-break` · `letter-spacing`, `word-spacing`, `line-height` · `vertical-align` · `text-align` & `text-indent`

---

#### Chapter 17 — CSS Background
> `chapter17.html` · `chapter17.css`

Covers all background-related properties and the powerful CSS gradient and sprite techniques.

**Concepts:** `background-color`, `background-image`, `background-repeat`, `background-size` (`cover`, `contain`), `background-position`, `background-attachment` (`fixed`, `scroll`) · Linear & radial gradients · Multiple backgrounds · CSS image sprites with hover effects

---

#### Chapter 18 — CSS Display
> `chapter18.html` · `chapter18.css`

Explains how elements are visually rendered and how display behavior can be changed.

**Concepts:** `display: block`, `inline`, `inline-block`, `none` · `visibility: hidden` vs `display: none` · `object-fit` (`fill`, `contain`, `cover`, `scale-down`) · Cursor types · `overflow` (`hidden`, `scroll`, `auto`) · `opacity`

---

#### Chapter 19 — CSS Layout
> `chapter19.html` · `chapter19.css`

The most comprehensive chapter — covers the full range of CSS layout mechanisms from classic to modern.

**Concepts:**
- **Position:** `static`, `relative`, `absolute`, `fixed`, `sticky`
- **Float & Clear:** `float: left/right`, `clear: both`, clearfix hack
- **Z-index:** Stacking contexts and layer ordering
- **Flexbox:** `flex-direction`, `justify-content`, `align-items`, `flex-wrap`, `gap`, `flex-grow`, auto-margin navigation
- **CSS Grid:** `grid-template-columns`, `grid-template-rows`, `grid-gap`, `grid-area`, named template areas

---

#### Chapter 20 — CSS Transitions & Animations
> `chapter20.html` · `chapter20.css`

Brings the page to life with time-based CSS effects — no JavaScript required.

**Transitions:** `transition-property`, `transition-duration`, `transition-timing-function` (`ease`, `linear`, `ease-in-out`, `cubic-bezier`), `transition-delay`

**Animations:** `@keyframes`, `animation-name`, `animation-duration`, `animation-iteration-count`, `animation-direction`, `animation-fill-mode`, `animation-play-state`

---

#### Chapter 21 — Advanced CSS
> `chapter21.html` · `chapter21.css`

Introduces powerful CSS features that reduce repetition and enable dynamic styling logic.

**Concepts:** CSS custom properties (`--variable-name`) · `var()` function · `calc()` for mixed-unit arithmetic · CSS counters (`counter-reset`, `counter-increment`, `content: counter()`)

---

#### Chapter 22 — Responsive Web Design
> `chapter22.html` · `chapter22.css`

Teaches the principles and techniques for building layouts that adapt to any screen size.

**Concepts:** Viewport meta tag · Fluid grids (percentage-based columns) · CSS media queries (`@media screen and (max-width: ...)`) · Breakpoints for mobile, tablet, and desktop · Responsive images · Responsive typography

## 🚀 Getting Started

No build tools, no package manager, no dependencies. This project runs entirely in the browser.

### Option 1 — Open Directly

```bash

git clone https://github.com/your-username/html5-css3-project.git

# Navigate into the project folder
cd html5-css3-project/1294236

# Open the portfolio homepage
open home.html          # macOS
start home.html         # Windows
xdg-open home.html      # Linux
```

Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension, then right-click `home.html` → **Open with Live Server** for instant hot-reload in the browser.
